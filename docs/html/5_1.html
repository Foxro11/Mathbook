Sumas de Riemman

Introducción

Las sumas de Riemann son un método para calcular el área bajo una curva. Imagina que tienes una curva que representa la función f(x) en el intervalo [a, b]. El área bajo la curva representa la integral definida de f(x) de a a b.
Las sumas de Riemann dividen el intervalo [a, b] en subintervalos y calculan el área de cada subintervalo. El área total bajo la curva se aproxima sumando las áreas de los subintervalos.

Tipos de Sumas de Riemann:

  • Sumas de Riemann izquierda: Se calcula el área de cada subintervalo usando el valor de la función en el extremo izquierdo del intervalo.
  • Sumas de Riemann derecha: Se calcula el área de cada subintervalo usando el valor de la función en el extremo derecho del intervalo.
  • Sumas de Riemann central: Se calcula el área de cada subintervalo usando el valor de la función en el punto medio del intervalo.
  • Sumas de Riemann trapezoidales: Se calcula el área de cada subintervalo como la mitad del área de un trapecio con bases en los valores de la función en los extremos del intervalo.

Error de aproximación:

  • La precisión de la aproximación de la suma de Riemann aumenta a medida que aumenta el número de subintervalos.
  • El error de aproximación se puede calcular utilizando la fórmula de error de Simpson.

Aplicaciones de las Sumas de Riemann:

  • Cálculo de áreas bajo curvas.
  • Cálculo de volúmenes de sólidos de revolución.
  • Cálculo de longitudes de curvas.
  • Aproximación de integrales definidas.

Ejercicios

Ejercicios Resueltos
  1. Ejercicio: Suma de Riemann trapezoidal

    2. Función: f(x) = e^x

    Intervalo: [0, 1]

    Número de subintervalos: 3

    Tipo de suma: Trapezoidal

    Solución:

    • Ancho de cada subintervalo: Δx = (b-a)/n = (1-0)/3 = 1/3
    • Puntos clave: x0 = a = 0, x1 = a + Δx = 1/3, x2 = a + 2Δx = 2/3, x3 = b = 1
    • Valor de la función en los puntos clave: f(x0) = 1, f(x1) = e^(1/3), f(x2) = e^(2/3), f(x3) = e
    • Área del primer rectángulo: A0 = f(x0)Δx/2 = 1 * (1/3) / 2 = 1/6
    • Área del segundo rectángulo: A1 = (f(x1) + f(x2))Δx/2 = (e^(1/3) + e^(2/3)) * (1/3) / 2
    • Área del tercer rectángulo: A2 = f(x3)Δx/2 = e * (1/3) / 2 = e/6
    • Suma de Riemann trapezoidal: SR = A0 + A1 + A2 = 1/6 + (e^(1/3) + e^(2/3)) * (1/3) / 2 + e/6

      • Resultado: La suma de Riemann trapezoidal es aproximadamente 1.84034.

    2. Función: f(x) = √x

    Intervalo: [0, 2]

    Número de subintervalos: 4

    Tipo de suma: Derecha

Formulario de Fórmulas

    Suma Izquierda:

    Σ (f(a + iΔx) * Δx), para i = 0, 1, 2, ..., n-1

    Suma Derecha:

    Σ (f(a + (i+1)Δx) * Δx), para i = 0, 1, 2, ..., n-1

    Suma Central:

    Σ (f(a + (i + 1/2)Δx) * Δx), para i = 0, 1, 2, ..., n-1

    Suma Trapezoidal:

    Σ ((f(a + iΔx) + f(a + (i+1)Δx)) * Δx / 2), para i = 0, 1, 2, ..., n-1

    Σ representa la sumatoria.

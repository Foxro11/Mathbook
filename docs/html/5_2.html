Integral Definida

Introducción

Las integrales definidas son una herramienta matemática fundamental para calcular el área bajo una curva, el volumen de un sólido de revolución, la longitud de una curva y muchas otras cantidades importantes en física, ingeniería y economía.
En términos simples, una integral definida calcula la suma de un número infinito de rectángulos infinitesimales de ancho constante que se aproximan a la curva bajo estudio.
El proceso de calcular una integral definida se llama integración.

Conceptos:

  • Función: La función a integrar, representada por f(x).
  • Intervalo de integración: El intervalo [a, b] donde se calcula la integral.
  • Límite de integración: Los valores a y b que definen el intervalo de integración.
  • Suma de Riemann: Una aproximación del área bajo la curva mediante rectángulos.
  • Valor de la integral: El límite de la suma de Riemann cuando el número de rectángulos tiende a infinito.
  • Teorema fundamental del cálculo: Este teorema establece una conexión entre la integración y la diferenciación, lo que permite calcular integrales definidas mediante antiderivadas.

Clasificación de Integral Definida :

  • Integrales impropias: Son integrales donde el intervalo de integración es infinito o la función presenta discontinuidades.
  • Integrales de Riemann-Stieltjes: Son una generalización de las integrales de Riemann que permiten integrar funciones con respecto a funciones no monótonas.

Aplicaciones en Diversas Áreas:

  • Cálculo de áreas y volúmenes: Se pueden usar para calcular el área bajo una curva, el volumen de un sólido de revolución o la longitud de una curva.
  • Cálculo de trabajo: Se pueden usar para calcular el trabajo realizado por una fuerza variable a lo largo de una distancia.
  • Cálculo de promedios: Se pueden usar para calcular el promedio de una función sobre un intervalo.
  • Cálculo de probabilidades: Se pueden usar para calcular la probabilidad de un evento en un espacio de probabilidad.

Ejercicios

Ejercicios Resueltos
  1. Ejercicio 1: Área bajo la curva

    2. Función: f(x) = x²

    Intervalo: [0, 1]

    Integral: ∫₀¹ x² dx

    Antiderivada: F(x) = (1/3)x³

    Resultado: F(1) - F(0) = 1/3

  2. Ejercicio 2: Volumen de revolución

    3. Función: f(x) = e^x

    Intervalo: [0, 1]

    Integral: π ∫₀¹ (e^x)² dx

    Resultado: π (e² - 1)

  3. Ejercicio 3: Longitud de curva

    4. Función: f(x) = √x

    Intervalo: [0, 1]

    Integral: ∫₀¹ √(1 + (1/2)x²) dx

    Resultado: 4/3

Formulario de Fórmulas

    Integrales definidas básicas
    • ∫₀^¹ f(x) dx
    • ∫_a^b f(x) dx
    • F(b) - F(a)
    Definiciones formales
    • ∫ f(x) dx = lim_(n→∞) Σ_(i=0)^(n-1) f(a + iΔx) Δx
    • ∫ f(x) dx = lim_(n→∞) Σ_(i=0)^(n-1) f(a + (i + 1/2)Δx) Δx
    • ∫ f(x) dx = lim_(n→∞) Σ_(i=0)^(n-1) (f(a + iΔx) + f(a + (i+1)Δx)) Δx / 2
    • ∫ f(x) dx = lim_(n→∞) Σ_(i=1)^n f(a + iΔx) Δx
    Ejemplos de integrales de funciones específicas
    • ∫ e^x dx = e^x + C
    • ∫ ln(x) dx = x ln(x) - x + C
    • ∫ sin(x) dx = -cos(x) + C
    • ∫ cos(x) dx = sin(x) + C

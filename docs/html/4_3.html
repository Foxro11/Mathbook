Limites

Introducción

El concepto de límite es fundamental en el cálculo diferencial. Un límite es el valor al que se aproxima una función cuando su variable independiente se acerca a un punto determinado. En este curso de cuarto semestre, nos enfocaremos en la definición formal de límite, diferentes tipos de límites y técnicas para calcularlos.

Explicación Detallada

Definición formal de límite:

El límite de la función \(f(x)\) cuando x se acerca a a, escrito como \(lim_(x->a) f(x)\), es \(L\) si para cualquier \(ε > 0\), existe un \(δ > 0\) tal que si \(0 < |x - a| < δ\), entonces \(|f(x) - L| < ε\).

Tipos de límites:

  • Límites finitos: El límite de la función es un número finito.
  • Límites infinitos: El límite de la función es infinito.
  • Límites laterales: El límite de la función es diferente cuando x se acerca a a por la derecha y por la izquierda.

Técnicas para calcular límites:

  • Sustitución directa: Si la función es continua en el punto a, entonces el límite es igual al valor de la función en el punto a.
  • Algebratización: Se simplifica la expresión de la función antes de calcular el límite.
  • L'Hôpital: Se utiliza cuando el límite es de la forma 0/0 o ∞/∞.
  • Comparación: Se compara la función con otras funciones cuyo límite sea conocido.

    Ejercicios

    Ejercicios Resueltos
    1. Calcula el límite \(lim_(x->2) (x² - 4x + 3)\).
        2. Solución: Sustituyendo directamente x = 2 en la función, obtenemos \(lim_(x->2) (x² - 4x + 3) = 2² - 4(2) + 3 = -1.\)
    2. Calcula el límite \(lim_(x->0) (sin(x) / x)\).
        3. Solución: Este límite es de la forma 0/0, por lo que se utiliza la regla de L'Hôpital. La derivada de \(sin(x)\) es \(cos(x)\) y la derivada de \(x\) es 1, por lo que \(lim_(x->0) (sin(x) / x) = lim_(x->0) (cos(x) / 1) = cos(0) = 1\).
    3. Calcula el límite \(lim_(x->∞) (x² / (x + 1))\).
        4. Solución: Dividiendo el numerador y el denominador por \(x²\), obtenemos \(lim_(x->∞) (x² / (x + 1)) = lim_(x->∞) ((x² / x²) / ((x + 1) / x²)) = lim_(x->∞) (1 / (1 + 1/x)) = 1\).
    Ejercicios Propuestos
    1. Calcula el límite \(lim_(x->3) √(x - 1)\).
    2. Calcula el límite \(lim_(x->π/2) tan(x)\).
    3. Calcula el límite \(lim_(x->∞) (e^x / x)).

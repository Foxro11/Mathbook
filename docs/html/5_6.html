Integración por Partes

Introducción

La integración por partes es una técnica fundamental en el cálculo integral que permite encontrar la integral de un producto de dos funciones. Se basa en la regla del producto para la diferenciación y es una herramienta poderosa para resolver integrales que no se pueden resolver mediante técnicas básicas.

Conceptos:

  • Integración: Es el proceso de encontrar la función original a partir de su derivada.
  • Integración por partes: Es una técnica para encontrar la integral de un producto de dos funciones.
  • Regla del producto: Es una regla de diferenciación que establece que la derivada del producto de dos funciones es igual a la primera función por la derivada de la segunda más la segunda función por la derivada de la primera.

Utilidad de las Integración por Partes:

  • La integración por partes se utiliza para resolver integrales que no se pueden resolver mediante las técnicas básicas de integración.
  • Es una herramienta poderosa que se puede utilizar para resolver una amplia gama de integrales.
  • Se utiliza en diversas áreas de la matemática, la física y la ingeniería.

Pasos Claves de la Integración por Partes:

  • Identificar las funciones u y v: El producto uv es la función que queremos integrar.
  • Elegir u: Es recomendable elegir u como la función que tiene una derivada más simple que v.
  • Calcular du y v: Calcular la derivada de u y la integral de v.
  • Aplicar la fórmula: Sustituir u, v, du y v en la fórmula de la integración por partes.
  • Resolver la integral: La integral original se reduce a dos integrales más simples. Resolver estas integrales para obtener la integral final.

Ejercicios

Ejercicios Resueltos

    La integración por partes es una técnica para encontrar la integral de un producto de dos funciones. A continuación se presentan algunos ejemplos de ejercicios resueltos utilizando esta técnica.

  1. Ejercicio: ∫ x * e^x dx

    2. Solución:

    1. Identificamos las funciones u y v: u = x, v = e^x
    2. Elegimos u: Elegimos u = x porque tiene una derivada más simple que v (du = dx).
    3. Calculamos du y v: du = dx, v = e^x
    4. Aplicamos la fórmula: ∫ uv dx = u ∫ v dx - ∫ u'v dx = x * e^x - ∫ e^x dx
    5. Resolvemos la integral restante: ∫ e^x dx = e^x + C
    6. Solución final: ∫ x * e^x dx = x * e^x - e^x + C

  2. Ejercicio: ∫ sin(x) * cos(x) dx

    3. Solución:

    1. Identificamos las funciones u y v: u = sin(x), v = cos(x)
    2. Elegimos u: Elegimos u = sin(x) porque tiene una derivada más simple que v (du = cos(x) dx).
    3. Calculamos du y v: du = cos(x) dx, v = sin(x)
    4. Aplicamos la fórmula: ∫ uv dx = u ∫ v dx - ∫ u'v dx = sin(x) * sin(x) - ∫ cos(x) * sin(x) dx
    5. Resolvemos la integral restante: ∫ cos(x) * sin(x) dx = -sin^2(x) + C (usando nuevamente integración por partes)
    6. Solución final: ∫ sin(x) * cos(x) dx = sin^2(x) - (-sin^2(x) + C) = 0 + C = C

  3. Ejercicio: ∫ ln(x) dx

    4. Solución:

    1. Identificamos las funciones u y v: u = ln(x), v = 1
    2. Elegimos u: Elegimos u = ln(x) porque tiene una derivada más simple que v (du = 1/x dx).
    3. Calculamos du y v: du = 1/x dx, v = x
    4. Aplicamos la fórmula: ∫ uv dx = u ∫ v dx - ∫ u'v dx = ln(x) * x - ∫ (1/x) * x dx
    5. Resolvemos la integral restante: ∫ (1/x) * x dx = x + C
    6. Solución final: ∫ ln(x) dx = ln(x) * x - x + C = x * ln(x) - x + C

    Estos son solo algunos ejemplos de integración por partes. Existen muchos otros tipos de integrales que se pueden resolver utilizando esta técnica.

Formulario de Fórmulas

    Fórmula básica

    ∫ uv dx = u ∫ v dx - ∫ u'v dx

    Identidades útiles
    • d(ln(x)) = 1/x dx
    • d(sin(x)) = cos(x) dx
    • d(cos(x)) = -sin(x) dx
    • d(tan(x)) = sec^2(x) dx
    • d(sec(x)) = sec(x) tan(x) dx

    Reglas de derivadas
    • Producto: (uv)' = u'v + uv'
    • Cociente: (u/v)' = (u'v - uv')/v^2
    • Función compuesta: (f(g(x)))' = f'(g(x)) * g'(x)

Ecuaciones lineales

Introducción

Una ecuación lineal es una ecuación de primer grado, es decir, una ecuación que tiene la forma \(ax + b = c\), donde \(a\), \(b\) y \(c\) son números reales y \(a\) no es igual a 0. La variable \(x\) es la incógnita que queremos determinar.

Explicación Detallada

Partes de una ecuación lineal:

  • Coeficiente: El número que acompaña a la variable.
  • Término independiente: El número que no tiene variable.
  • Solución: El valor de la variable que hace que la ecuación sea verdadera.

Métodos para resolver ecuaciones lineales:

  • Método de despeje: Se despeja la variable en la ecuación y se opera para obtener su valor.
  • Método de balanceo: Se suman o restan cantidades iguales a ambos lados de la ecuación para obtener la variable sola.
  • Método gráfico: Se representa la ecuación en una gráfica y se busca el punto de intersección con el eje \(x\).

Ejercicios

Ejercicios Resueltos
  1. Ejercicio 1: Resuelve la ecuación \(2x + 5 = 11\).
      2. Solución: \(2x + 5 = 11\)
      \(2x = 6\)
      \(x = 3\)
  2. Ejercicio 2: Resuelve la ecuación \(3x - 2 = 4\).
      3. Solución: \(3x - 2 = 4\)
      \(3x = 6\)
      \(x = 2\)
Ejercicios Propuestos
  1. Ejercicio 1: Una tienda vende un producto por $10. Si Ana compra 3 productos, ¿cuánto debe pagar?
  2. Ejercicio 2: Un tren viaja a una velocidad de 80 km/h. Si recorre 240 km, ¿cuánto tiempo ha tardado?

A destacar

  • Ecuación lineal: \(ax + b = c\)
  • Solución: \(x = \frac{{c - b}}{a}\)

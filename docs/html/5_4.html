Integración por Cambio de Variable

Introducción

La integración por cambio de variable es un método para encontrar la integral de una función compuesta.
En términos simples, este método consiste en sustituir una parte de la función a integrar por una nueva variable, de tal manera que la integral se vuelve más simple de resolver.
El proceso de integración por cambio de variable se basa en la regla de la cadena para las derivadas.

Conceptos:

  • Función compuesta: Una función de la forma f(g(x)), donde f y g son funciones.
  • Nueva variable: La variable que se introduce para sustituir una parte de la función a integrar.
  • Sustitución: El proceso de reemplazar la parte de la función a integrar por la nueva variable.
  • Diferencial: La diferencial de la nueva variable, que se utiliza para ajustar la integral.

Utilidad de las Integración por Cambio de Variable:

  • Integrar funciones que no se pueden integrar directamente.
  • Simplificar la integración de funciones complejas.

Pasos Claves de la Integración por Cambio de Variable:

  • Identificar la función compuesta: Se busca una función u(x) tal que f(x)=g(u(x)).
  • Definir la nueva variable: Se define u=u(x) como la nueva variable.
  • Calcular la diferencial: Se calcula du=u ′(x)dx.
  • Sustituir en la integral: Se sustituye f(x) por g(u(x)) y dx por du.
  • Resolver la integral: Se resuelve la integral utilizando la nueva variable u.
  • Revertir la sustitución: Se sustituye u por u(x) en la integral resultante.

Ejercicios

Ejercicios Resueltos
  1. Ejercicio 1: Integral de x^2 √(x+1)

    2. Función: f(x) = x^2 √(x+1)

    Solución: Integración por cambio de variable

    Pasos:

    1. Identificar la función compuesta: f(x) = x^2 y u(x) = √(x+1).
    2. Definir la nueva variable: u = √(x+1).
    3. Calcular la diferencial: du = 1/(2√(x+1)) dx.
    4. Sustituir en la integral: ∫ x^2 √(x+1) dx = ∫ u^2 du.
    5. Resolver la integral: ∫ u^2 du = u^3/3 + C.
    6. Revertir la sustitución: u^3/3 + C = (√(x+1))^3/3 + C.

    Resultado: ∫ x^2 √(x+1) dx = (√(x+1))^3/3 + C

Formulario de Fórmulas

    Fórmula general

    ∫ f(u(x))u'(x) dx = F(u(x)) + C

    Fórmula con función inversa

    ∫ f(x) dx = ∫ f(u(u^(-1)(x)))u'(u^(-1)(x)) dx

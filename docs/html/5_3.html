Integral Indefenida

Introducción

Las integrales indefinidas son una herramienta matemática fundamental para encontrar la antiderivada de una función.
En términos simples, la integral indefinida de una función f(x) es una familia de funciones F(x) tal que F ′(x)=f(x).
El proceso de encontrar la integral indefinida de una función se llama integración.

Conceptos:

  • Función: La función a integrar, representada por f(x).
  • Antiderivada: Una función F(x) tal que F ′(x)=f(x).
  • Constante de integración: Una constante arbitraria que se añade al resultado de la integración.
  • Notación: La integral indefinida de f(x) se denota por ∫f(x)dx.

Utilidad de las Integrales Indefinidas:

  • Encontrar el área bajo una curva: La integral indefinida de f(x) entre a y b se puede usar para calcular el área bajo la curva de y=f(x) entre x=a y x=b.
  • Calcular el volumen de un sólido de revolución: La integral indefinida de f(x) se puede usar para calcular el volumen del sólido de revolución generado al girar la curva de y=f(x) alrededor del eje x.
  • Resolver ecuaciones diferenciales: Las integrales indefinidas se pueden usar para resolver ecuaciones diferenciales de primer orden.

Metodos para Encontrar las Integrales Indefinidas:

  • Integración por inspección: Se basa en reconocer la función a integrar como la derivada de una función conocida.
  • Integración por partes: Se utiliza para integrar el producto de dos funciones.
  • Integración por sustitución: Se utiliza para integrar una función compuesta.
  • Integración por fracciones parciales: Se utiliza para integrar funciones racionales.

Ejercicios

Ejercicios Resueltos
  1. Ejercicio: Integral de x^2

    2. Función: f(x) = x^2

    Solución: Integración por inspección

    Resultado: ∫ x^2 dx = x^3/3 + C

  2. Ejercicio: Integral de e^x

    3. Función: f(x) = e^x

    Solución: Integración por inspección

    Resultado: ∫ e^x dx = e^x + C

  3. Ejercicio: Integral de sin(x)

    4. Función: f(x) = sin(x)

    Solución: Integración por inspección

    Resultado: ∫ sin(x) dx = -cos(x) + C

Formulario de Fórmulas

    Integrales indefinidas básicas
    • ∫ x^n dx = x^(n+1)/(n+1) + C, donde n ≠ -1
    • ∫ e^x dx = e^x + C
    • ∫ sin(x) dx = -cos(x) + C
    • ∫ cos(x) dx = sin(x) + C
    Integrales de funciones especiales
    • ∫ 1/x dx = ln|x| + C
    • ∫ √x dx = 2/3 x^(3/2) + C
    • ∫ ln(x) dx = x ln(x) - x + C
    Integrales de funciones trigonométricas inversas
    • ∫ sin^(-1)(x) dx = -x√(1-x^2) + C
    • ∫ cos^(-1)(x) dx = x√(1-x^2) + C
    • ∫ tan^(-1)(x) dx = 1/2 ln|1+x^2| + C

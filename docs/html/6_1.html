Distribución de Frecuencia

Introducción

La distribución de frecuencia es una herramienta estadística que permite organizar y resumir datos numéricos, mostrando la frecuencia con la que aparecen ciertos valores o rangos de valores dentro de un conjunto de datos. Es una herramienta fundamental para comprender la distribución de la información y visualizar las tendencias en los datos.

Tipos de Distribuciones de Frecuencia:

  • Distribución de frecuencia simple: Se utiliza para datos categóricos o cualitativos, como colores, preferencias, etc. La tabla de frecuencia simple muestra la cantidad de veces que aparece cada categoría.
  • Distribución de frecuencia agrupada: Se utiliza para datos numéricos. Los datos se agrupan en intervalos y se registra la frecuencia de cada intervalo. La tabla de frecuencia agrupada muestra los intervalos, la frecuencia absoluta (número de datos en cada intervalo), la frecuencia relativa (frecuencia absoluta dividida por el total de datos) y la frecuencia acumulada (suma de las frecuencias relativas hasta el intervalo actual).

Utilidad de las Distribuciones de Frecuencia:

  • Encuestas: Se puede utilizar para analizar las respuestas a preguntas de opción múltiple.
  • Censos: Se puede utilizar para analizar la distribución de la población por edad, sexo, etc.
  • Control de calidad: Se puede utilizar para analizar la distribución de defectos en un producto.
  • Estudios de mercado: Se puede utilizar para analizar las preferencias de los consumidores.

Elementos de una Distribución de Frecuencia:

  • Variable: La característica que se está midiendo.
  • Valores: Los diferentes valores que puede tomar la variable.
  • Frecuencia: La cantidad de veces que aparece cada valor.
  • Intervalos: Rangos de valores que se agrupan en la distribución de frecuencia agrupada.
  • Frecuencia absoluta: Número de datos en cada intervalo.
  • Frecuencia relativa: Frecuencia absoluta dividida por el total de datos.
  • Frecuencia acumulada: Suma de las frecuencias relativas hasta el intervalo actual.

Formulario de Fórmulas

    Cálculo de Frecuencias

    • Frecuencia absoluta (FA): FA = # datos en cada intervalo
    • Frecuencia relativa (FR): FR = FA / Total datos
    • Frecuencia acumulada (FAC): FAC = FA + Frecuencia acumulada del intervalo anterior

    Medidas de Posición

    • Media (X̅): X̅ = Σx / n
    • Mediana (Me): Me = Valor divide distribución en 2 partes con igual número de datos
    • Moda (Mo): Mo = Valor con mayor frecuencia

    Medidas de Dispersión

    • Rango (R): R = Valor máximo - Valor mínimo
    • Desviación media (DM): DM = Σ|x - X̅| / n
    • Varianza (V): V = Σ(x - X̅)² / n
    • Coeficiente de variación (CV): CV = (DM / X̅) * 100%

    Gráficos

    • Histograma: Representa frecuencia de cada intervalo con barras.
    • Diagrama de caja y bigotes: Muestra distribución a través de 5 valores: mínimo, Q1, mediana, Q3, máximo.
